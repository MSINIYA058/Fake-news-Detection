# Fake-news-Detection
Data Science Internship

## Project Description
This project identifies whether a news article is real or fake using Natural Language Processing and Machine Learning.

## Technologies used
-Python
-Pandas
-NLTK
-Scikit-Learn

## Features
-Text Processing
-Feature Extraction
-Model Training
-Fake News Classification

## Outcome
The model classifies news articles as real or fake.
